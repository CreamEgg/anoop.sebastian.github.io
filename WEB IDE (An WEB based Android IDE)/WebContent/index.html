WEBIDEA, Web based IDE for Android development

WEBIDEA provides easy access to a set of preconfigured Android development tools, allowing the user to focus solely on the application development.
No setups, No Installations, No Configurations

Start WEBIDEA About WEBIDEA

// What You'll Get

  • Feature #1 Zero Setup

    No enviornment configurations/installations are required to begin Android development.

  • Feature #2 Start Coding From Day 1

    Beginning Android development is as easy as choosing a project name, target and few other properties and clicking Create Project button.

  • Feature #3 More Time For Learning

    Comparing to other desktop based IDEs, WEBIDEA takes less that half the amount of time to create and compile projects.

  • Feature #4 Reduced Learning Curve

    The learning curve for Android development has been significantly affected by the implementation of WEBIDEA.

  • Feature #5 Real-Time Log Capture From User's Android Device

    The Android logs are captured in real-time and displayed in the web interface.

  • Feature #6 No Hardware Connections Between The Computer And The Android Device

    The Android device is notified by the server using Google Cloud Messsaging, when a new APK file is available to download.

// Testimonials

  • Tester 1 No software installations, No installations were needed to develop my Hello World Android App.

  • Tester 2 Less powerful computer, The only software required to use WEBIDEA was a standard web browser, a less powerful computer was required for the development process.
